# Student Info Project

## Description
The Student Info Project is designed to manage student information seamlessly. It allows for the addition, removal, and display of student details, making it easier for educators and administrators to maintain student records.

## Functionality
The `Student_info.java` file includes the following functionalities:
- **Add Student**: Allows users to input and store student details.
- **Remove Student**: Provides the ability to remove a student from the records.
- **Display Students**: Lists all students currently in the system with their details.

## Usage
To use the Student Info Project:
1. Clone the repository: `git clone https://github.com/thebkofficial-dev/Developers_Arena`
2. Navigate to the project directory: `cd Developers_Arena/Student_info_project`
3. Compile the Java file: `javac Student_info.java`
4. Run the program: `java Student_info`
